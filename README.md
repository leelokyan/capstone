# capstone

## Setup Instructions:
1. Download Node.js (https://nodejs.org/en/)
2. Pull repository
3. cd capstone
4. npm install

## Download New Node.JS Module:
1. cd capstone
2. npm install "module_name" --save
3. Commit package.json and package-lock.json

## Start Server:
1. node server.js
2. Launch browser at localhost:3000
