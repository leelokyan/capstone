# capstone

## Setup Instructions:
1. Download Node.js (https://nodejs.org/en/)
2. Pull repository
3. 
'''
cd capstone
npm install
'''

## Download New Node.JS Module:
1. 
'''
cd capstone
npm install "module_name" --save
'''
2. Commit package.json and package-lock.json

## Start Server:
1. 
'''node server.js'''
2. Launch browser at localhost:3000
