USERS:
	Add New User:
		- Endpoint : /add_user
		- Request Type : POST
		- Request:
			- String : fname
			- String : lname
			- String : email (email is the same as userId)
			- String : workspacePassword
		- Response:
			- Boolean : success (-1 on failure)
			- String : error (null on failure)

	Get User:
		- Endpoint : /get_user
		- Request Type : POST
		- Request : 
			- String : email
			- string:workspaceId
			- string:authCode
		- Response :
			- String : fname
			- String : lname
			- String : email
			- String : error

STRATEGY:
	Create Strategy
		- Endpoint : /create_strategy
		- Request Type : POST
		- Request:
			- String : strategyName
			- String : description
		- Response:
			- string : strategyId (id or "" on error)
			- String : error (null if no error)

	Update Strategy
		- Endpoint : /update_strategy
		- Request Type : POST
		- Request:
			- string : strategyId
			- string : name 
			- String : description
		- Response:
			- bool : success
	
	Get Strategies
		- Endpoint : /get_strategy
		- Request Type : POST
		- Request:
		- Response:
			- Object [] : myStrategyList (or empty list)
				- string : strategyId
				- String : strategyName
				- String : description

OBJECTIVE:
	Get My Objectives
		- Endpoint : /get_my_objectives
		- Request Type : POST
		- Request: 
			- string : userId
		- Response :
			- object [] : objectives
				- string: objectiveID
				- string: name
				- string: startDate
				- string: endDate
				- string: tags
				- string []: assignedUsers
				- string: description
				- string: goal (parent goal)
				- bool : complete
	Get Objectives of Goal
		- Endpoint : /get_objectives
		- Request Type : POST
		- request :
			- string : goalId
		- response :
			- bool : success
			- string : error
			- object [] : objectives
				- string: objectiveID
				- string: name
				- string: startDate
				- string: endDate
				- string: tags
				- string []: assignedUsers
				- string: description
				- string: goal (parent goal)
				- bool : complete

	Create Objective 
		- Endpoint : /create_objective
		- request type : POST
		- request :
			- string : goalId
			- string[] : tags
			- string : userId
			- string : startDate
			- string : endDate
		- response :
			- string : objectiveId
			- bool : success
			- string : error

	Update Objective
		- endpoint : /update_objective
		- request type : POST
		- request : 
			- string : objectiveId
			- string [] : tags
			- string [] : assignedUsers
			- string : startDate
			- string : endDate
			- int : status
		- response :
			- bool : success
			- string: error

	Update Objective Status
		- endpoint : /update_objective_status
		- request type : POST
		- request :
			- string : objectiveId
			- int : status
		- response :
			- bool : success
